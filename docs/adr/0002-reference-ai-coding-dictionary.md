# Reference the AI Coding Dictionary

Use Matt Pocock's AI Coding Dictionary as the external vocabulary reference for AI-coding concepts: every semantically matching occurrence in prose remains in canonical English and links to its dictionary entry instead of duplicating definitions locally. Reuse the dictionary's seven section names, with attribution, as the closed tag vocabulary for Learn pages; author or agent review decides semantic matches, so the build must not rewrite text through blind keyword replacement. Apply links in paragraphs, lists, tables, and captions, but not in headings, code, URLs, Mermaid sources, or SVG text; links keep native same-tab browser behavior.
