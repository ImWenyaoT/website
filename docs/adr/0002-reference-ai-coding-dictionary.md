# Reference the AI Coding Dictionary

Use Matt Pocock's AI Coding Dictionary as the external vocabulary reference for AI-coding concepts: every semantically matching occurrence in body prose remains in canonical English and links to its dictionary entry instead of duplicating definitions locally. Repeated occurrences in the same paragraph or section are all linked; mixed Chinese-English phrases are acceptable because visible canonical vocabulary and Wikipedia-like outbound definitions take priority over minimizing link density. Reuse the dictionary's seven section names, with attribution, as the closed tag vocabulary for Learn pages; author or agent review decides semantic matches, so the build must not rewrite text through blind keyword replacement. Apply links in paragraphs, lists, tables, and captions, but not in H1-H6 headings, code, URLs, Mermaid sources, or SVG text; links keep native same-tab browser behavior.
